Get Started

How To Install Repository ?

  • Go to File Manager From Settings.
  • Click on Add Sources.
  • Use https://henry-richard.ml/Kodi Repos as a path and give it a name Kodi Repos Henry.

  • Go to Add-ons from settings.
  • Click Install From Zip File.
  • Select Kodi Repos Henry >> zips >> repository.henry >> repository.henry-x.x.x.zip
